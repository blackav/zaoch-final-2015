Есть полоска из N клеточек, каждая клеточка покрашена в свой цвет. Далее содержатся Q запросов, каждый из них: перекрась клетку x_i в цвет c_i. После выполнения каждого запроса необходимо говорить минимальное количество непересекающихся одноцветных полосок, которыми можно покрыть данную. Проще говоря, количество нерасширяемых одноцветных отрезков.

Группа 1. N, Q <= 1000
Группа 2. N, Q <= 100,000
