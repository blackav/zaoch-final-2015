Проводятся выборы с выборщиками в K туров. В первом туре участвую N человек. На каждом этапе они разбиваются на группы равного размера, в каждой из групп производится голосование. Для выхода в следующий этап нужного выборщика необходимо строгое большинство. Вопрос, сколько людей необходимо подкупить на начальном этапе, чтобы выиграть выборы?

divs - максимальное число делителей у числа до maxn
pairs - максимум суммы числа делителей по делителям числа до maxn

1) k = 2, n = 100              (перебрать разбиения n = a * b)
2) k <= 10, n <= 10^3          (n^2 * k / тупой перебор)
3) k <= 10, n <= 10^6          (n*k / умный перебор)
4) k <= 10, n <= 10^11         (divs^2 * k + sqrt(n))
5) k <= 10, n <= 10^13         (divs^2 + pairs * k + sqrt(n))
6) k <= 10, n <= 10^15         (pairs * k + sqrt(n))
